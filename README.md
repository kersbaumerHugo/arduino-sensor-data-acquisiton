# BAJA SAE - Electronics and Embedded Systems
Este repositório contém os códigos e projetos relacionados à equipe BAJA SAE Alibabaja, focados no desenvolvimento dos sistemas eletrônicos e embarcados para os veículos off-road da competição.
## Sobre o projeto
O BAJA SAE é uma competição internacional em que equipes universitárias projetam, constroem e competem com veículos off-road. Nossa equipe desenvolve os sistemas eletrônicos embarcados que controlam diversos aspectos do veículo, como sensores, atuadores, e interfaces de controle.
## Minhas contribuições
- Desenvolvimento de firmware para microcontroladores Arduino, incluindo leitura de sensores e controle de atuadores.
- Criação de placas de circuito impresso (PCB) para integração dos sistemas eletrônicos.
- Organização das conexões elétricas e documentação técnica dos sistemas.
- Implementação de códigos para monitoramento e diagnóstico dos sistemas embarcados.
## Tecnologias utilizadas
- Linguagem C/C++ para programação Arduino
- Eagle
- Git para controle de versão
## Como usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/kersbaumerHugo/baja-electronics.git
   ```
2. Abra os códigos na IDE Arduino ou outra compatível.
3. Siga as instruções específicas em cada pasta para compilação e upload.
## Estrutura do repositório
- `/firmware` - códigos fonte para microcontroladores
- `/pcb` - arquivos de design das placas eletrônicas
- `/docs` - documentação técnica e diagramas elétricos
