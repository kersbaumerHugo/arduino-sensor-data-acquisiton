# BAJA SAE Alibabaja - Electronics and Embedded Systems
This repository contains the code and projects related to the BAJA SAE Alibabaja team, focused on the development of electronic and embedded systems for the off-road vehicles used in the competition.
## About the project
BAJA SAE is an international competition where university teams design, build, and race off-road vehicles. Our team develops the embedded electronic systems that control various aspects of the vehicle, such as sensors, actuators, and control interfaces.
## My contributions
- Firmware development for Arduino microcontrollers, including sensor reading and actuator control.
- Creation of printed circuit boards (PCBs) for integrating electronic systems.
- Organization of electrical connections and technical documentation of the systems.
- Implementation of code for monitoring and diagnostics of embedded systems.
## Technologies used
- C/C++ programming for Arduino
- Eagle
- Git for version control
## How to use
1. Clone this repository:
   ```bash
   git clone https://github.com/kersbaumerHugo/baja-electronics.git
   ```
2. Open the code in the Arduino IDE or another compatible IDE.
3. Follow the specific instructions in each folder for compilation and upload.
## Repository structure
- `/firmware` - source codes for microcontrollers
- `/pcb` - electronic board design files
- `/docs` - technical documentation and electrical diagrams

# BAJA SAE Alibabaja - Electronics and Embedded Systems
Este repositório contém os códigos e projetos relacionados à equipe BAJA SAE Alibabaja, focados no desenvolvimento dos sistemas eletrônicos e embarcados para os veículos off-road da competição.
## Sobre o projeto
O BAJA SAE é uma competição internacional em que equipes universitárias projetam, constroem e competem com veículos off-road. Nossa equipe desenvolve os sistemas eletrônicos embarcados que controlam diversos aspectos do veículo, como sensores, atuadores, e interfaces de controle.
## Minhas contribuições
- Desenvolvimento de firmware para microcontroladores Arduino, incluindo leitura de sensores e controle de atuadores.
- Criação de placas de circuito impresso (PCB) para integração dos sistemas eletrônicos.
- Organização das conexões elétricas e documentação técnica dos sistemas.
- Implementação de códigos para monitoramento e diagnóstico dos sistemas embarcados.
## Tecnologias utilizadas
- Linguagem C/C++ para programação Arduino
- Eagle
- Git para controle de versão
## Como usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/kersbaumerHugo/baja-electronics.git
   ```
2. Abra os códigos na IDE Arduino ou outra compatível.
3. Siga as instruções específicas em cada pasta para compilação e upload.
## Estrutura do repositório
- `/firmware` - códigos fonte para microcontroladores
- `/pcb` - arquivos de design das placas eletrônicas
- `/docs` - documentação técnica e diagramas elétricos
